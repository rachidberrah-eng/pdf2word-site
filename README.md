# محوّل PDF إلى Word (رياضيات/فيزياء)

موقع بسيط: يرفع المستخدم PDF، يختار شكل العنوان والنص،
والموقع يولّد ملف Word نهائي بمعادلات حقيقية قابلة للتعديل
(ليست صورًا)، باستخدام Gemini API لتنظيم المحتوى وتحويل
المعادلات إلى LaTeX، ثم pandoc لتوليد Word.

## التشغيل محليًا (للتجربة فقط)
```
pip install -r requirements.txt
export GEMINI_API_KEY="ضع_مفتاحك_هنا"
python app.py
```
افتح: http://localhost:5000

انظر دليل النشر المرفق مع المحادثة.
